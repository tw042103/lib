**命令模式快捷键**
| 模式 | 命令 | 描述 |
| ---- | ---- | --- |
| 命令模式 | `i` | 在当前光标位置进入输入模式 |
| 命令模式 | `a` | 在当前光标位置之后进入输入模式 |
| 命令模式 | `I` | 在当前行的开头进入输入模式 |
| 命令模式 | `A` | 在当前行的结尾进入输入模式 |
| 命令模式 | `o` | 在当前光标下一行进入输入模式 |
| 命令模式 | `O` | 在当前光标上一行进入输入模式 |
| 输入模式 | `esc` | 在任何情况下输入`esc`都能回到命令模式 |
